Preguntas
frecuentes

¿Qué puedo hacer en este sitio web?

En este sitio web puedes: 1. Generar un color que contraste perfectamente con un fondo específico. 2. Generar combinaciones de colores para fondo y texto. 3. Generar colores balanceados con la regla 60-30-10.

¿Cómo utilizo los generadores de colores?

¡Es muy fácil! Solo haz clic en los recuadros de arriba para acceder a cada generador.

¿Por qué debería utilizar un generador de color accesible?

Para garantizar que el contenido sea fácil de leer y accesible para todos. Además ¡es una fuente de inspiración para crear combinaciones únicas y funcionales!

¿Para qué sirve el generador de un solo color?

Te ayuda a encontrar un color ideal que contraste con un fondo específico. Es perfecto cuando ya tienes definido el color del fondo.

¿Para qué sirve el generador de dos colores?

Te permite crear combinaciones aleatorias de colores para fondo y texto. Es perfecto para explorar nuevas combinaciones de colores y encontrar la más adecuada para tus diseños.

¿Para qué sirve el generador de la regla 60-30-10?

Te permite crear paletas equilibradas y balanceadas mediante 3 colores: primario (60%), secundario (30%) y acento (10%). Es perfecto para lograr un diseño visualmente equilibrado.
